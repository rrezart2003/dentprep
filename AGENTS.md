# AGENTS.md

## Project
DentPrep – a dental exam preparation application.

## Goal
Help dental students study through:
- exam-style questions
- flashcards
- clinical case scenarios

## Main sections
1. Exams – multiple choice questions with similar answer options.
2. Flashcards – quick revision cards for concepts.
3. Clinical cases – patient scenario questions (to be expanded later).

## Important rules
- Do not break existing navigation.
- Do not change database structure unless necessary.
- Prefer small incremental changes.
- Maintain clean UI and readable question layout.

## Code style
- Keep components modular.
- Avoid large refactors unless explicitly requested.

## Development philosophy
Focus on:
1. learning effectiveness
2. realistic exam questions
3. clean and simple UX
